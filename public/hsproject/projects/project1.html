Why am I making this website?

     I am making this website because I think it's fun to experiment with coding and I've always wanted to have a solid understanding of web development. After starting this project and realizing how much fun designing websites is, I am considering a future in web design. I will also be exploring other job opportunities in the computer science field throughout the rest of this semester. Even if I don't choose web design as my career, the skills I have learned will be very useful if I ever start a business and need to make a website for it.

How did I make this website?

     The first thing I did was research popular website designs and I made a sketch of how I wanted to design mine. Although I heavily changed the design, it provided me with a good place to begin learning. I made this website using mostly HTML and CSS. These are two of the most essential coding languages used in web development. HTML is used to structure and add basic things to the page, while CSS is used to style those things and make the webpage unique. HTML was much easier to get the hang of than CSS because CSS has many different modifiers that are hard to remember without looking up.

What will I put on this website in the future?

     I will be showcasing the rest of my POL projects and I will be constantly improving and changing the website design where I see fit.

What did I learn while making this website?

     I gained an intermediate knowledge of website development in CSS and HTML. I hope to teach myself javascript next so I can have more interactive things on my website. I also gained insight on the way that developers need to think to solve problems.

How much time have I spent designing this website?

     As of Januray 26, 2017, I have spent about 15 hours in total learning HTML and CSS and about 15-20 hours actually coding and designing the website. It was really easy to get lost in my project because I like the challenge.

What are the advantages and disadvantages of a web design career?

     Web design is certainly a fun and interesting career, but of all the jobs in the computer science field, web design is one of the lowest paying. Regardless, this is a passion of mine and I will leave it open as an option for the future.
