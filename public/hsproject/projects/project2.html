Explaining the Code

     The first step I took towards designing my website was to research website design theory. Web design theory refers to the layout of the website and what the viewers will like or dislike. Since this is my first website, I went with a simple design consisting of a logo, a navigation bar, and a dropdown menu.

     Once I had a design in mind, the next step was to learn the most basic and important programming language used in web design, HTML. This language is used to structure a website and add very basic, unembellished things to the page. The next language I had to learn was CSS. This language was a little bit harder to learn because there were a lot more commands than HTML. CSS builds onto the HTML code by adding formatting, color, positioning, etc. CSS is what sets a basic website apart from something beautiful. For example, here are a few side-by-side pictures of my website with the CSS coding included and with the CSS coding removed:



     This shows how important CSS is in creating a modern website. CSS is normally applied using a separate file called a "stylesheet." This stylesheet is linked to the page and contains all of the CSS coding that can be referenced at anytime in the main HTML file.

     After tens of hours of research, I had an extensive knowledge of both of these languages and began making my website. With a combination of these languages I was able to make a fully functional basic website with menus, animations, text, and other things.

     I have attached some code from my website below. You can hover over it for an explanation on what each line does. Many lines of the CSS code have been removed and only one page of my website is shown as the focus of this page is to explain the things visible in the Presentation 1 page only.

     Once all the coding for the website was completed, the next step was to find a company to host the website. There are two components needed in order to bring a website online, a hosting server and a domain name. For the hosting server I went with a basic free one from 000webhost.com. For the domain name I chose davebecker.xyz because .xyz domains are very cheap compared to the popular ones like .com and .net.
